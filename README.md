## 使用方式

每天只做三件事：

1. 补一个知识点。
2. 写一段能运行的代码或实验。
3. 记录一个可复述的面试答案。

不要只收藏资料。这个方向的竞争力来自能解释真实报文路径、能写转发代码、能用抓包和 Linux 工具定位问题。
